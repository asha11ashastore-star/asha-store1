# Save Your Hero Image Here

## Steps to Add Your Bridal Saree Image:

1. **Save the image you uploaded as:**
   - Filename: `hero-saree.jpg`
   - Location: This folder (same folder as this README)
   - Full path: `/Users/divyanshurathore/shopall/frontend/customer-website/public/images/hero-saree.jpg`

2. **How to save it:**
   - Right-click the image (the one you just uploaded)
   - Select "Save Image As..."
   - Navigate to this folder
   - Save as `hero-saree.jpg`

3. **After saving:**
   - Refresh your browser at http://localhost:3001
   - The new image should appear as the homepage hero!
